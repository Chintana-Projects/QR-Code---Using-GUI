# QR Code Generator using GUI 

This project is a QR Code Generator application built using Python's tkinter library for the GUI and qrcode library for generating QR codes.

It allows users to create custom QR codes with various options:
- Choose QR code type (Text, URL, Phone Number, Email)
- Enter data for the QR code
- Specify save location
- Select fill color and background color
- Adjust QR box size
- Choose shape style (Square or Circle)

The generated QR code is saved as a PNG image at the specified location.

Libraries Used:  

- tkinter: For creating the GUI.
- qrcode: For generating QR codes.
- PIL (Pillow): For image processing and drawing.
